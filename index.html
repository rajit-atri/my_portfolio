Digital Stopwatch Using 555 Timer

Undergraduate Project

Designed and implemented a digital stopwatch circuit capable of counting from 0 to 59 seconds using a 555 Timer IC in astable mode. The timer generated precise 1 Hz clock pulses, which were processed through two cascaded counter ICs. The counters fed outputs into BCD-to-7 segment decoders to display elapsed seconds on seven-segment displays.

Key Methods / Tools

  • 555 Timer IC (astable mode)
  • Counter ICs (cascaded) (MQTT/HTTP)
  • BCD-to-7 Segment Decoder
  • Digital Electronics
  • Circuit Simulation & Prototyping

Highlights

  • Successfully built a functional hardware prototype capable of accurate second-by-second counting.
Project Report

Implementing a Buck Converter to Enhance Power Management in RC Cars

Undergraduate Project

Built a low-cost Bluetooth-controlled RC car and integrated a buck converter to step down battery voltage for the motor/control circuits. The goal was to improve runtime, reduce heat, and stabilize supply rails so the car maintains consistent speed and responsiveness as the battery discharges. Control signals are sent from a mobile app to the onboard receiver and motor driver ICs for precise movement.

Key Methods / Tools

  • Buck converter (step-down DC-DC)
  • DC motors and Motor driver ICs
  • Power budgeting (efficiency & thermal checks)
  • Chassis assembly ICs
  • Circuit Simulation & Prototyping

Highlights

  • Converts higher battery voltage (e.g., 7.4–12 V) down to optimal motor/logic rails (e.g., 5 V/3.3 V) with 85–95% efficiency.
Project Report

Designing a 1-Bit 4-to-1 Multiplexer Using 1-Bit 2-to-1 Multiplexers

Undergraduate Project

Designed a digital circuit for a 1-bit, 4-to-1 multiplexer (MUX) by combining multiple 1-bit, 2-to-1 MUX units. The project demonstrates how complex circuits can be built from simpler components to optimize data routing. Cadence Virtuoso was used to create the schematic and simulate the design, including transient analysis, DRC, and LVS checks.

Key Methods / Tools

  • Cadence Virtuoso
  • 2-to-1 MUX circuit design
  • Schematic entry
  • Transient analysis
  • DRC (Design Rule Check) & LVS (Layout vs. Schematics)

Highlights

  • Showcased hierarchical digital design by constructing a 4-to-1 MUX from simpler 2-to-1 blocks.
Project Report

Designing the Physical Layout of a 2-to-1 Multiplexer and Verifying with DRC, ERC, and LVS

Undergraduate Project

Developed the physical layout of a 2-to-1 multiplexer (MUX) using industry-standard EDA tools as an extension of the earlier schematic design. The project focused on translating logical design into a physical circuit layout while ensuring compliance with fabrication constraints. Verification was carried out through Design Rule Check (DRC), Electrical Rule Check (ERC), and Layout vs. Schematic (LVS) to validate accuracy and functionality.

Key Methods / Tools

Cadence Virtuoso, Physical design workflows, DRC, ERC, LVS, Layout design
  • Cadence Virtuoso
  • Physical design workflows
  • DRC (Design Rule Check), LVS (Layout vs. Schematics) & ERC (Electrical Rule Check)
  • Layout design

Highlights

  • Designed a fabrication-ready physical layout of a 2-to-1 MUX.
